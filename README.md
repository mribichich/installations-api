# installations-api
Api to save installations

## Requerimientos

nodejs 8.x
yarn

## Usage

### Install dependencies

`yarn`

### Development

```bash
yarn start:watch
yarn test:watch
```

### Production

```bash
NODE_ENV=production yarn build
NODE_ENV=production yarn start
```
